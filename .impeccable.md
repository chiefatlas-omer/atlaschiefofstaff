## Design Context

### Users
Internal Atlas Growth team (founders, sales reps, CS managers) using the Command Center daily to manage tasks, review call intelligence, track coaching, and monitor business outcomes. They're busy operators who need information fast — no time for visual clutter. The desktop overlay app serves the same users during their workflow (email, CRM, Slack) via voice commands.

### Brand Personality
**Clean. Calm. Professional.**
Atlas is not flashy — it's the quiet system that makes everything work. Think Apple's restraint with Linear's precision. The product should feel like a trusted executive assistant: organized, reliable, and always one step ahead.

### Emotional Goal
**"I'm in control."** Every screen should make the user feel organized, informed, and confident that nothing is slipping through the cracks. The data should be clear at a glance — no hunting, no confusion, no visual noise.

### Aesthetic Direction
- **Reference:** Linear — clean layout, beautiful typography, purposeful whitespace, data presented with care
- **Theme:** Light mode primary (clean, professional). Dark mode for desktop overlay only.
- **Typography:** System font stack (SF Pro / Segoe UI / system-ui). Clean hierarchy: bold headings, medium labels, regular body.
- **Anti-references:** ChatGPT (too much empty space), Salesforce (too cluttered), Jira (too dense and ugly)
- **Motion:** Subtle and purposeful only. No gratuitous animations. Smooth transitions for panel reveals and state changes.

### Design Principles
1. **Clarity over decoration** — Every element earns its place. If it doesn't help the user make a decision or take action, remove it.
2. **Hierarchy through typography, not color** — Use font weight and size to create hierarchy. Reserve color for status and actions only.
3. **Density with breathing room** — Show enough data to be useful without overwhelming. Linear-style spacing: generous but not wasteful.
4. **Consistent patterns** — Cards look like cards everywhere. Badges look like badges everywhere. One pattern per concept.
5. **Professional by default** — The UI should feel like it was designed by a real design team, not generated by AI. No rainbow gradients, no emoji overuse, no gimmicks.

### Technical Stack
- **Web:** React 19 + Tailwind v4 + Vite 8 (no component library — custom components)
- **Desktop:** Electron + vanilla CSS (glassmorphic overlay panels)
- **Fonts:** System font stack (no web fonts loaded)
- **Icons:** None currently — text/emoji only
