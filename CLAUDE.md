# Atlas Chief of Staff

AI-powered Chief of Staff for Atlas Growth — Slack bot, web command center, and desktop voice assistant.

## Design Context

### Brand Personality
**Clean. Calm. Professional.** Think Apple's restraint with Linear's precision. The product should feel like a trusted executive assistant: organized, reliable, and always one step ahead.

### Brand Colors
| Token | Hex | Role |
|-------|-----|------|
| `brand-primary` | `#4F3588` | Primary actions, active states |
| `brand-secondary` | `#5A3C9E` | Hover states, secondary accents |
| `brand-light` | `#F3F1FC` | Light surfaces, subtle backgrounds |
| `brand-pale` | `#FAF9FE` | Near-white, card backgrounds |
| `accent-pink` | `#EC4899` | Section labels, highlights |
| `accent-orange` | `#FB923C` | Secondary highlights |
| `accent-violet` | `#A78BFA` | Informational elements |
| `success` | `#22C55E` | Completed, positive |
| `warning` | `#EAB308` | Processing, attention |
| `error` | `#EF4444` | Overdue, failures |

### Emotional Goal
**"I'm in control."** Every screen = organized, informed, confident. Data clear at a glance.

### Design Principles
1. **Clarity over decoration** — every element earns its place
2. **Hierarchy through typography, not color** — font weight/size for hierarchy, color for status only
3. **Density with breathing room** — Linear-style: generous but not wasteful
4. **Consistent patterns** — one pattern per concept across all pages
5. **Professional by default** — feels designed by a real team, not AI-generated

### Theme
- **Web:** Light mode (clean, professional, Linear-inspired)
- **Desktop overlay:** Dark glassmorphic purple (keep as-is)

### Reference
- Linear (clean layout, typography, whitespace)
- Anti-references: ChatGPT (too empty), Salesforce (too cluttered), Jira (too dense)
